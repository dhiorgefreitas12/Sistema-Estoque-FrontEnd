# 📦 Controle de Estoque Frontend

Este é o **Frontend** do projeto **Controle de Estoque**, feito em **Vue 3 + Vite + TypeScript**.

---

## 🚀 Tecnologias

- Vue.js 3
- Vite
- TypeScript
- Axios

---

## ⚙️ Instalação

```bash
# Clone o repositório
git clone <URL_DO_REPO>

# Entre na pasta do projeto
cd controle-estoque

# Instale as dependências
npm install
```
